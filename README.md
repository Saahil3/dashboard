# Dashboard Application

This project is a responsive dashboard built using **React/Next.js** and styled with **Tailwind CSS**.

---

## Features

### Core Features
- **Responsive Perfect Implementation**: Matches the Figma design specifications.
- **Interactive Table**: Displays data with sorting and pagination functionality.
- **Mock Data**: Populates the dashboard with realistic mock data.

### Bonus Features
- **Table Sorting**: Sort table data by columns dynamically.
- **Pagination**: Navigate through data across multiple pages.

---

## Installation and Setup

### Prerequisites
- Node.js (>=16.x)
- npm or yarn

### Installation Steps
1. Clone the repository:
   ```bash
   git clone <repository_url>
   cd dashboard-application
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```


## Technologies Used

- **React.js**
- **Tailwind CSS** (for styling)
